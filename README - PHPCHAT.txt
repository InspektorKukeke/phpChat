In ordet to test phpChat you have couple of options:

1. Locally

 * Requirements
	* Project code (zip file)
	* Xampp (Apache + mysql)
 * Steps
	* Start Apache and MySQL services in XAMPP Control Panel
	* Unzip project code into Xampp htdocs folder: (example) C:\xampp\htdocs
	* Path to index.php file should be similar to C:\xampp\htdocs\phpChat\index.php
	* Ensure that phpmyadmin user is "root" and pw is empty
	* Open phpmyadmin panel on http://localhost/phpmyadmin/
	* Create new database "chatroomdb"
	* Run SQL script file "chatroomdb" from project root folder
	* Start website on your browser - http://localhost/phpChat/index.php
	* Either register new account or use admin account: username: "peeter", password: "test1234"
	* Enjoy the website:)
